# REQ-001: Server Connection

**ID:** REQ-001
**Domain:** Connection
**Status:** Draft
**Priority:** P0 (MVP)

## Description

The application MUST connect to a Snapcast server via WebSocket on port 1704 (default).

## Acceptance Criteria

- [ ] Connect using IP address or hostname
- [ ] Discover servers on local network via mDNS/Bonjour
- [ ] Save multiple connection profiles
- [ ] Auto-connect to last used server on startup
- [ ] Show connection status (connected/disconnected/connecting)

- Protocol: JSON-RPC over WebSocket

- Connection timeout: 10 seconds
