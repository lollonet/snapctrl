# REQ-001: Server Connection

**ID:** REQ-001
**Domain:** Connection
**Status:** Done
**Priority:** P0 (MVP)

## Description

The application MUST connect to a Snapcast server via TCP on port 1705 (default control port).

## Acceptance Criteria

- [x] Connect using IP address or hostname
- [x] Discover servers on local network via mDNS/Bonjour
- [ ] Save multiple connection profiles
- [x] Auto-connect to last used server on startup
- [x] Show connection status (connected/disconnected/connecting)

## Technical Notes

- Protocol: JSON-RPC over raw TCP sockets
- Default port: 1705 (control port)
- Connection format: `tcp://{host}:{port}`
- Connection timeout: 10 seconds
