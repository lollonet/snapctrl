# REQ-009: System Theme Support

**ID:** REQ-009
**Domain:** UI/UX
**Status:** Draft
**Priority:** P1 (Important)

## Description

The application MUST follow the system theme (light/dark mode).

## Acceptance Criteria

- [ ] Detect system theme at startup
- [ ] Switch theme when system theme changes
- [ ] Light theme with good contrast
- [ ] Dark theme with good contrast
