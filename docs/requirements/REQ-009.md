# REQ-009: System Theme Support

**ID:** REQ-009
**Domain:** UI/UX
**Status:** Done
**Priority:** P1 (Important)

## Description

The application MUST follow the system theme (light/dark mode).

## Acceptance Criteria

- [x] Detect system theme at startup
- [x] Switch theme when system theme changes (runtime, macOS dark mode toggle)
- [x] Light theme with good contrast (22-color ThemePalette)
- [x] Dark theme with good contrast (22-color ThemePalette)

## Implementation Notes

- ThemeManager singleton in `src/snapcast_mvp/ui/theme.py`
- ThemePalette dataclass with 22 named color fields
- Uses Qt 6.5+ `colorScheme()` API for auto-detection
- Replaced ~40 hardcoded hex `setStyleSheet` calls across 7 UI files with palette lookups
- [PR #13](https://github.com/lollonet/snapctrl/pull/13)
