# REQ-006: Client Details View

**ID:** REQ-006
**Domain:** Display
**Status:** Draft
**Priority:** P1 (Important)

## Description

The application MUST display detailed information for each client.

## Acceptance Criteria

- [ ] Client name (editable)
- [ ] Host IP/hostname
- [ ] MAC address
- [ ] Connection status (online/offline)
- [ ] Current volume and mute state
- [ ] Latency value (read-only for MVP)
- [ ] Snapclient version
