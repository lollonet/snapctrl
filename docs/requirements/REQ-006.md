# REQ-006: Client Details View

**ID:** REQ-006
**Domain:** Display
**Status:** Done
**Priority:** P1 (Important)

## Description

The application MUST display detailed information for each client.

## Acceptance Criteria

- [x] Client name (editable via context menu rename)
- [x] Host IP/hostname
- [x] MAC address
- [x] Connection status (online/offline) with color coding
- [x] Current volume and mute state
- [x] Latency value (interactive QSpinBox for connected clients, read-only for disconnected)
- [x] Snapclient version
- [x] System info (OS and architecture)
- [x] Last seen timestamp with human-readable format
- [x] Network RTT ping measurement with color-coded thresholds
- [x] Server-side jitter stats (median, P95, samples) with Âµs precision
