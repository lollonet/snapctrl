# REQ-010: System Tray Integration

**ID:** REQ-010
**Domain:** UI/UX
**Status:** Done
**Priority:** P1 (Important)

## Description

The application MUST provide a system tray icon.

## Acceptance Criteria

- [x] Minimize to tray instead of closing (close-to-tray behavior)
- [x] Tray icon shows connection status (green/red dot overlay via QPainter)
- [x] Right-click menu: Show/Hide, Quit
- [x] Quick volume controls for each group (embedded QWidgetAction slider)
- [x] Mute All / Unmute All

## Implementation Notes

- SystemTrayManager in `src/snapctrl/ui/system_tray.py`
- Show/Hide window toggle (double-click or menu)
- Group status entries with average volume percentages
- Now Playing metadata from sources
- Debounced menu rebuild on state changes (500ms)
