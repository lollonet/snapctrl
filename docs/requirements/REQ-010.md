# REQ-010: System Tray Integration

**ID:** REQ-010
**Domain:** UI/UX
**Status:** Draft
**Priority:** P1 (Important)

## Description

The application MUST provide a system tray icon.

## Acceptance Criteria

- [ ] Minimize to tray instead of closing
- [ ] Tray icon shows connection status
- [ ] Right-click menu: Show/Hide, Quit
- [ ] Quick volume controls for each group
- [ ] Mute All / Unmute All
